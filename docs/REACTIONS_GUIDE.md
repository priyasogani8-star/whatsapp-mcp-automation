# 😍 Reactions Handler - Send Animated Reactions

Make messages more fun with animated reactions!

## What Are Reactions?

Reactions are emoji responses to messages with **animated effects**.

### Types of Reactions:
- 🎉 **Confetti** - Colorful confetti animation
- 🎆 **Fireworks** - Fireworks effect
- ✨ **Sparkles** - Sparkle animation
- ❤️ **Hearts** - Floating hearts
- 😂 **Emoji** - Regular emoji reactions

## How to Use

### Send Reaction
```json
{
  "messageId": "msg_123",
  "userId": "user_1",
  "emoji": "🎉",
  "effectType": "confetti",
  "intensity": "heavy"
}
