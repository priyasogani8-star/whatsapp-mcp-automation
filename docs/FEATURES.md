## 🗳️ Poll Voting Tool

**Create polls with two types:**

### Single-Choice Polls
- User picks exactly 1 option
- Can change vote anytime
- Perfect for: "Which restaurant?" "Yes or No?"

### Multiple-Choice Polls
- User picks multiple options (1-3+)
- Can add/remove selections
- Perfect for: "What properties?" "Which skills?"

**Example:**
```
Create poll: "Where should we eat?"
- Pizza
- Sushi
- Burger

User votes: Pizza
User changes to: Sushi

Results:
- Pizza: 0 votes
- Sushi: 1 vote
- Burger: 0 votes
```

## 📢 Group Mentions

**Send important notifications to groups**

### Mention Types:
- @all - Mention everyone
- @role - Mention by role (e.g., "Admins")
- @specific - Mention specific people
- @admin - Mention admins only

### Features:
- Create group roles
- Send role-based mentions
- Track notification delivery
- See who read notification
- Statistics on mentions

**Example:**
```
Create role:
- Role: "Team Leads"
- Members: Alice, Bob, Charlie

Send mention:
- Message: "Important update needed!"
- Type: @role "Team Leads"
- Notification level: High

Tracking:
- Sent to: 3 people
- Delivered: 3
- Read by: 2 people (Alice, Bob)
- Pending: 1 person (Charlie)
```
